# Contributed Scripts

This directory is for existing bisection scripts and tools from
contributors. Drop your scripts here as-is — we will refactor them
into reusable components as the project matures.

## How to contribute

1. Create a subdirectory with your name or organisation
2. Add your script(s) and a short README explaining:
   - What the script does
   - What it depends on (tools, infrastructure, APIs)
   - How to run it
3. Open a pull request
